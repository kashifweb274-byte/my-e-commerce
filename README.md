My E-Commerce Project 🛒
Ye ek full-stack e-commerce project hai jisme Frontend (React) aur Backend (Express/Node.js) dono shamil hain.

Features
Backend: Express server ka istemal karke API banayi gayi hai.

CORS: Custom middleware lagaya gaya hai taaki frontend aur backend aasani se connect ho sakein.

Dummy Data: Project ko test karne ke liye locally data handle kiya gaya hai.

Tech Stack
Frontend: React.js, CSS

Backend: Node.js, Express.js

Database: (Agar aapne MongoDB use kiya hai toh yahan mention karein)
